# RELATÓRIO DO PROJETO - PC MANAGER

## Descrição

O PC Manager é um sistema web desenvolvido em Laravel para o gerenciamento de peças de computador. O sistema permite realizar o cadastro, edição, visualização, listagem e exclusão de componentes, facilitando o controle de estoque.

## Objetivo

O objetivo do projeto foi aplicar os conceitos estudados na disciplina de Desenvolvimento de Aplicações com Laravel, utilizando a arquitetura MVC e os principais recursos do framework.

## Funcionalidades Implementadas

* Cadastro de peças
* Listagem de peças
* Visualização de detalhes
* Edição de peças
* Exclusão de peças
* Seeders para inserção de dados iniciais

## Tecnologias Utilizadas

* PHP 8.3
* Laravel 13
* SQLite
* Blade
* Vite
* Composer

## Estrutura do Projeto

O projeto foi organizado seguindo o padrão MVC do Laravel:

* Models
* Controllers
* Migrations
* Seeders
* Rotas Resource
* Views Blade

## Banco de Dados

O banco de dados foi criado utilizando Migrations do Laravel e populado com dados iniciais através de Seeders.

## Dificuldades Encontradas

Durante o desenvolvimento foram encontrados alguns problemas relacionados à configuração inicial do projeto, criação das migrations e organização das rotas. Todos foram corrigidos durante a implementação.

## Resultado

O sistema foi concluído com sucesso, permitindo o gerenciamento completo das peças cadastradas e demonstrando a utilização prática dos recursos oferecidos pelo framework Laravel.
